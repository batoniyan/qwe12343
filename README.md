# 可以做什么
托管闲置宿主机，可将宿主机切割成**LXC容器或者KVM虚拟机**。当前可以实现LXC容器的**IPV4 NAT、IPV4 NAT & IPV6（独立分配）、IPV4 NAT & IPV6 NAT（共享宿主机）、IPV6 Only、IPV6 NAT**以及KVM虚拟机的**IPV4 NAT、IPV4 NAT & IPV6（独立分配）、IPV6 Only**这些网络模式的分配。可上架成付费实例到托管市场，可不公开仅自己使用，可通过好友功能分享仅好友间共享。

# 添加节点
系统推荐：Debian12+、Ubuntu 22.04+，配置要求最低1C1G10G（1核心1G实际运行内存10G硬盘空间）
